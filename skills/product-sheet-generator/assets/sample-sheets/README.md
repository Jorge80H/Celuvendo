# Sample Product Sheets

This directory contains example product specification sheets that demonstrate the correct format and structure.

## Available Samples

Reference the actual completed product sheets in:
`../../Fichas_Productos/`

### Examples to Reference:

1. **TECNO-Camon-40.md**
   - Standard format example
   - Mid-range 4G device
   - Demonstrates all required sections
   - Shows proper Spanish marketing copy
   - Example of charger inclusion notes

2. **TECNO-Camon-40-Pro.md**
   - Premium model example
   - Demonstrates 4G vs 5G variant handling
   - Shows advanced camera specs documentation

3. **SAMSUNG-Galaxy-A06.md**
   - Entry-level example
   - Demonstrates pricing competitiveness alerts
   - Shows "cargador no incluido" warnings
   - Example of budget bundle strategies
   - Margin analysis sections

## Key Features Demonstrated

- Complete technical specifications in organized tables
- Dual-language terminology (Spanish/English)
- Colombian market pricing context
- Competitive comparison format
- Sales strategy recommendations
- Bundle and cross-selling opportunities
- E-commerce-specific notes
- FAQ sections addressing real customer concerns

## How to Use These Samples

When creating a new product sheet:
1. Use the template from `references/product-sheet-template.md`
2. Reference these samples for tone, structure, and completeness
3. Adapt sections based on product tier (entry-level vs premium)
4. Maintain consistent formatting and emoji usage
5. Ensure all critical warnings are clearly marked

## Notes

- All prices are in Colombian Pesos (COP)
- Marketing copy is in Spanish
- Technical terms use standard English abbreviations
- Sources are documented for pricing and specs
- Critical information (charger, water resistance) is highlighted with emojis and warnings
